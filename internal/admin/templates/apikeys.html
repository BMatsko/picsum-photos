Usage

Upload an image via the API:

curl -X POST {{.RootURL}}/api/v1/upload \
  -H "Authorization: Bearer <your-key>" \
  -F "photo=@/path/to/image.jpg" \
  -F "author=Jane Doe" \
  -F "tags=nature,portrait"

Returns 201 JSON with the assigned id, dimensions, and tags. See Docs for the full reference.
