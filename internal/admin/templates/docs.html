API Reference

Base URL: {{.RootURL}}

Random image

GET /[width]/[height]

Returns a different random image on every request. Not cacheable.

ExampleResult
{{.RootURL}}/200/300200×300 random image
{{.RootURL}}/400/400400×400 square random image

Seeded image

GET /seed/[seed]/[width]/[height]

The same seed always returns the same photo. Seed can be any string. Cached for 1 day.

ExampleResult
{{.RootURL}}/seed/hello/400/300Always the same photo for "hello"
{{.RootURL}}/seed/user-123/200/200Per-user stable avatar

Image by ID

GET /id/[id]/[width]/[height]

Fetch a specific photo by ID. Cached for 1 day.

Example
{{.RootURL}}/id/1/400/300

Query parameters

ParameterExampleEffect
?grayscale/200/300?grayscaleConvert to grayscale
?blur/200/300?blurBlur with default amount (5)
?blur=1–10/200/300?blur=2Blur with custom intensity
