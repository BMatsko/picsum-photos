How it works

/seed/abc/400/300 Resolves (abc, —) → same image every time
/seed/abc/400/300?tag=nature Resolves (abc, nature) → nature-filtered image, stored independently
/seed/abc/400/300?tag=urban Resolves (abc, urban) → different image, its own slot
/seed/abc/400/300?tag=nature again Returns the original nature image — slot memory is permanent
