Seeds

Seeds are deterministic — the same seed string always returns the same photo. Hash algorithm: murmur3.

How seeding works

Any string can be a seed. The seed is hashed with murmur3 into an int64, which seeds a Go rand.New(), and Intn(len(photos)) picks the photo. The result is stable as long as the order of photos (sorted by ID) does not change.
